Description

This is a collection of screenshots I took inside of Minecraft: Java Edition. I wanted to include these in my portfolio because I think it can be exciting constantly testing the boundaries of video games. By using a combination of different mods, shaders, and quality settings, I was able to capture some visually stunning scenes of the landscape in Minecraft.

SERVER SETUP

I have utilized many different Minecraft server hosting services in the past, such as Apex Hosting, Bisect Hosting, and Shockbyte. However, I prefer PebbleHost the most. This service provides users with an incredible console view that contains a lot of crucial information when it comes to hosting a multiplayer Minecraft server.

PebbleHost also offers much cheaper options than other services I tried, and is arguably one of the best paid server options out there. This service makes it easy for me to integrate mods, scheduled tasks, and backups for my servers. It also allows for easy file management from other tools like FileZilla.
